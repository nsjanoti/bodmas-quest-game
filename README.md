# BODMAS Quest – Interactive Learning Game

An interactive educational web game designed to teach and reinforce the BODMAS (Order of Operations) rule using visual step-by-step problem solving.

This project demonstrates frontend-backend integration using HTML, CSS, JavaScript, PHP, and MySQL.

---

## Project Objective

The goal of this game is to help students:

- Understand operator precedence (BODMAS rule)
- Develop logical reasoning skills
- Learn step-by-step expression solving
- Engage in active learning instead of passive memorization

Players are shown a mathematical expression and must select which operation should be solved next, according to BODMAS rules.

---

## What is BODMAS?

BODMAS stands for:

- B – Brackets  
- O – Orders (Powers/Roots)  
- D – Division  
- M – Multiplication  
- A – Addition  
- S – Subtraction  

This rule defines the correct order for solving mathematical expressions.
